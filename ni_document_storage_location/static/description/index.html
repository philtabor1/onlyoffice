Document Storage Location

This module allows you to store documents at any location.

  • Once you have install the module go to Document Storage App --> Document Type, If the document has Expiration Date? then checked the Expiration date.


  • Now go to Document Storage App --> Location, set the directory path for the documents to be store.


  • Now you will have smart button know as 'Documents' on Contacts as well on Employees.


  • Click on it and upload the documents and select store location.


  • Now you can see the uploaded documents at the store location.
